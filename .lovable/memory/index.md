# Project Memory

## Core
MinusFlow ToolKit (MinusFlow.net) — multi-platform media toolkit. BG #0f0f0f, cards #1a1a1a. Red accent. Downloads branded MinusFlow.net_ prefix.
Supabase Edge Functions as backend proxies. No puppeteer — HTTP scraping only.
Platforms: YouTube, TikTok, Instagram (active). Facebook, Douyin (planned).
